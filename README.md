## Interactive Personal Blog Platform

- This is a simple blog app built with HTML , CSS, and JavaScript. It lets users create, edit, and delete short posts. All posts are saved using localStorage.

# Project Files
- images
- index.html
- styles.css
- script.js

# How to Run
- Open index.html in your brower.

# Reflection
- Building this app helped me understand how JavaScript interacts with the DOM.
At first, getting the edit and delete buttons to work correctly was a challenge,
especially while keeping everything synced with localStorage.

I solved most issues by testing my code in CodePen, where I could instantly see results and fix errors faster.
It made experimenting and debugging much easier before moving my final code into VS Code.

I also learned how to break problems into smaller parts and test each step before adding new features.
